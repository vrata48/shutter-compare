# Shutter Compare

A browser-based tool for comparing specific regions of photographs taken with different camera settings (aperture, shutter speed, ISO).

## Features

- Load JPEG and Canon CR2 RAW files
- Select up to 5 regions to compare across all images
- View EXIF data: Aperture, Shutter Speed, ISO, Focal Length, Exposure Compensation, Camera Model
- Automatic EV (exposure value) difference calculation between images
- Histogram overlay for each crop
- Hide/show images for focused comparison
- Adjustable zoom (10-500%)
- No server required - runs entirely in browser

## Use Cases

- Compare lens sharpness at different apertures
- Evaluate noise levels across ISO settings
- Find optimal exposure in bracketed shots
- Test focus accuracy across frames

## Usage

1. Open index.html in a browser
2. Drag & drop or select images
3. Draw selection regions on the reference image
4. Compare crops with EXIF data and histograms
